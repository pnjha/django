To activate virtual environment for python development

$sudo pip install virtualenv 
$ mkdir ~/Dev
$ cd ~/Dev
$ mkdir venv && cd venv
$ virtualenv -p python3.6 .
$ source bin/activate

To deactivate:

(venv) $ deactivate

To reactivate:

$ cd /path/to/your/virtual/env/
$ source bin/activate

To setup django project 

$ pip install django
$ mkdir src && cd src
$ django-admin.py startproject cfehome .
$ cd cfehome
$ python manage.py migrate
$ python manage.py createsuperuser 

To see all the python pakages installed 
pip freeze

To runserver:
python manage.py runserver

To run all of django stuff from python shell
python manage.py shell 

To get all the objects created for Product class
Product.objects.all()

To create Product class objects from python shell
Product.objects.create(title='lenovo',description='chinnes',price='25000')
